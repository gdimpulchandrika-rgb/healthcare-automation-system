# AI-Powered Healthcare Appointment & Patient Reminder System

## Project Overview
This project automates healthcare appointment booking, patient data storage, and confirmation email notifications using n8n workflow automation.

## Features
- Patient appointment form
- Automated email confirmation
- Google Sheets database integration
- Workflow automation using n8n

## Technologies Used
- n8n
- Google Sheets
- Gmail API
- Workflow Automation

## Workflow
Patient Form → Google Sheets → Gmail Notification

## Use Case
This system helps hospitals and clinics reduce manual appointment management and improve patient communication.
